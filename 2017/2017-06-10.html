2. Mac 随想录（ Hello Mac ）

发表于 2017-06-08 | 分类于 Mac 随想录 | | 阅读次数
上一篇有的没的瞎扯了一些东西，最后终于把名字定下来了，从这一篇起我开始正式介绍我所认识的 Mac！并开始尝试为每一篇文章定一个清晰的结构，用最少的语言把想表达的问题说清楚，准备好了吗？

认识 Mac

好了能看到这里的，相信你已经拥有了或者打算拥有一台 Mac 让我们从打开包装开始吧！

2016 之后带 Touch bar 版本的 Mac 默认打开盖子后电脑会自动开机，不要怀疑是不是自己误触了开机键还是被别人用过的机器！这是系统的标定，从第一次打开 Mac 它就在提醒你自己可以随时待命，相比于市面上绝大多数 PC 机不使用时需要关机以及慢悠悠的开机过程，Mac 开盖及用的这种操作很酷！在 Mac 上你不用担心不关机电脑会发烫，会过度损耗电池，现在我可以告诉你担心是多余的，Mac 有着优秀的休眠策略和电源管理技术，即使你把电脑不关机放上好几天，下次打开时它照样精力充沛。

好了，现在应该到了初始化设置的界面，一般都是选择语言和设置键盘，连接 WiFi 设置锁屏密码等选项，不用着急，跟着指示走，相信这难不倒你。其中需要强调的是 Apple ID 很重要！！！如果你用过 Apple 其他设备，如果你以前不太重视保护自己的 Apple ID， 那么从现在开始一定要牢记自己的 Apple ID！密码最好复杂一点，自己比较健忘的话，最好记在其他地方，或者使用密码管理软件（1Password是个不错的选择，以后我会抽时间推荐一些 Mac 上优秀的 App ）还有就是打开两步验证，为什么要这么重视 Apple ID 呢？应为 Apple ID 就相当于 Apple 给你发的身份证！你使用 Apple 产品的时时刻刻都需要这个 ID 这是 Apple 生态里最重要的东西，不仅仅是一个验证，更是链接所有 Apple 产品的枢纽，如果你有很多 Apple 的产品，你会惊讶这带来的便利，如果自己的 Apple ID 落在了坏人手里，他就可以利用这个远程锁定你的设备，进行勒索，窃取你同步到 iCloud 的资料等，这样会很棘手！所有为了不让自己陷入窘境，还是提早防范，不要抱侥幸心理！
