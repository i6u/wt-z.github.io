结婚这件事

发表于 2017-05-03 | 分类于 随想 | | 阅读次数
生活疲惫不堪，理想衣不遮体，什么时候开始，我们变的顺从

最近有两个朋友结婚，祝福他们！

  毕业以前，结婚这件事离我们很遥远，就是一件只会发生的亲戚朋友身上的一件事，很少有人主动聊起这个事，多数人如你我一样，在象牙塔下对异性的所有憧憬都在爱情里，男女朋友腻在一起就好，有父母的地方就是家，户口本上都是血肉亲，我们依在父母之后。那个时候还可以任性的把自己比作一个孩子，毕业遥遥无期。

  不知走了多久，我们走到了毕业季，恍惚间，自己已经不再年轻，我们甚至还没有准备好就被赶上了人生的下一趟旅行，战战兢兢，背起行囊开始接受社会的洗礼。独自面对生活，一切都是那么变化多端，没有人提前告诉我们该读什么书，该走什么路，上课要注意听讲，考试前要认真复习，一天二十四小时全有我们自己安排。突如其来的自由让我们惊慌失措，我们开始不习惯曾经自己梦寐以求的自由，二十多年的应试教育告诉我们，选择题比填空题要简单的多！我们喜欢做简单的事，我们需要适当的束缚，需要有人告诉自己该做什么。于是毕业后我们做出了人生中第一个重大的决定，找一份固定的工作！从学校到公司，从课堂到办公室，从一个框到另一个框。没有人问自己为什么要这么做，只是看着大家都这么做，自己照着做而已。

  告别老师，走出校门，带着父母的嘱咐和对家的思念，来到他乡，钱成了我们最稀缺的东西，我们天天熬夜，拼命学习，想尽办法赚钱，却不知道赚多少钱才是个够。在自己奋斗的城市有一套房一直以来都是一个奢望，多数人成了少数人的工具！见风使舵，唯利是图成了这个社会的主旋律，我们被迫活的很累却无能为力。我们努力往上爬，丝毫不敢怠慢。

  我们自己走的都那么辛苦，未来的路都没有想好去哪里，怎么忍心赌上别人的青春，就像我姐对我说的，不要轻易结婚，不要轻易离婚。

  婚姻是一辈子的事，一定要找到一个对的人，然后告诉全世界，呐，我就要和这个人过一辈子，从青春到白头。等久一点又有何妨，好好打磨自己，期待下个转角遇到幸运，不要着急，35 岁之前你有的时间
